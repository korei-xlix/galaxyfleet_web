新しくゲームを始める New Game
新しくゲームを始めます。
ユーザID、ユーザ名、パスワードを入力してください。
ユーザID：ログインに必要なIDキーです。（半角英数字 8文字以上）
ユーザ名：ゲーム内に表示される司令官名を指定できます。（全角16文字まで）
パスワード：ログインに必要なパスワードです。（12文字以上）
　パスワードに使用可能な文字　半角英数字 < > ( ) { } ! ? @ + - * / = % # & $
全て入力して[START]を押すと、ゲームが始まります。
Start a new game.
Please enter your user ID, user name and password.
User ID: An ID key required for login. (8 or more single-byte alphanumeric characters)
User name: You can specify the commander name displayed in the game. (Up to 8 double-byte characters)
Password: The password required to log in. (12 characters or more)
　Characters that can be used for passwords　Half-width alphanumeric characters < > ( ) { } ! ? @ + - * / = % # & $
Enter all and press [START] to start the game.
