ゲームのはじめ方 :: How to start the game
新規でゲームを始めます。まずこの画面でユーザ登録をしてください。
ユーザID、ユーザ名、パスワードを入力してください。
  • ユーザID

    ログインに必要なIDキーです。（半角英数字 8文字以上）

  • ユーザ名

    ゲーム内に表示される司令官名を指定できます。（全角16文字まで）

  • パスワード

    ログインに必要なパスワードです。（12文字以上）
    　　パスワードに使用可能な文字　半角英数字 < > ( ) { } ! ? @ + - * / = % # & $

全て入力したあと、規約３箇条に同意（チェックボックスのチェック）してください。
規約を読む（リンククリック）することで各チェックボックスが有効になります。

全てチェックし、[START]を押すと、ゲームが始まります !!

Start a new game. First, register as a user on this screen.
