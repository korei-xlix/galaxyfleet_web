ゲームのはじめ方 :: How to start the game
新しくゲームを始める場合は[NEW]を押してください。
途中からゲームを再開するには、[Select Save File]でセーブデータを読んでください。
そのあとユーザID、パスワード、パスコードを入力して、[Login]を押してください。
Press [NEW] to start a new game.
To restart the game from the middle, read the save data with [Select Save File].
Then enter your user ID and password and passcode and press [Login].
ログアウトした場合は、下のセーブデータと、パスコードを必ず保存してください。
なおユーザID、パスワード、パスコードを忘れたり、セーブデータを保存し忘れた場合、
データの復元は不可能です。
If you log out, be sure to save the save data below and your passcode.
If you forget your user ID, password, passcode, or forget to save the save data,
Data cannot be restored.
