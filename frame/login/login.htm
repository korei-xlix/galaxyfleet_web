＜How to save save data＞
Since the web browser does not have the ability to save save data,
Follow the procedure below to save the save data.
1. Copy the save data above.
2. Create a new text file in an appropriate folder.
3. Open file 2 with Notepad etc.
4. Paste and save.
　　※It is easier to manage if you add a date to the file name.
